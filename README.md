# poc-minimal-node-and-express

&nbsp;

### Install Dependencies

```
npm install
```

&nbsp;

### Start locally

```
npm run start
```

Goto http://localhost:3000

&nbsp;

### Deploy

```
npm run build
```

Creates `dist/` folder containing `package.json`, `package-lock.json` and `index.js` ready to deploy to the cloud
